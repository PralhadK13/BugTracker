Steps to install bug-resolve

i) Install ember-cli
npm install -g ember-cli

ii) Create your project
ember new bug-resolve

iii) Go to bug-resolve project and replace project containing app folder with this app.

iv) Now go to  bug-resolve in terminal 
cd bug-resolve

v) Run bug-resolve
ember server

vi) Now navigate to http://localhost:4200 in the browser to see bug resolve application
